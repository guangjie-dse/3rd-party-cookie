# miniSPA
Minimal demo SPA of [auth0.js](https://auth0.com/docs/libraries/auth0js/v9) SDK
Pure html+javascript based

## How to Setup
* If you use Python 2

    `$ python -m SimpleHTTPServer 3000`

* If you use Python 3

    `$ python -m http.server 3000`

* If you use Nodejs

    `$ npm install`

    `$ node server.js`
